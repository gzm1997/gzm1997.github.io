sysu教务选课系统登陆

准备：
python3
安装tesseract,本人提供的tesseract百度云下载，亲测版本4比较多坑，这是版本3，已经够用的了
链接: http://pan.baidu.com/s/1boHp2Jt 密码: 6trj

  • 下载本项目到本地
  • 进入主目录，输入以下命令安装相关依赖：
pip install -r requirements.txt

  • 在如下59和64行输入你的账号和密码：

  • 输入一下命令运行代码：

    python login.py
